Making revisions in LateX

I recently revised a manuscript using easyReview. I found this super useful. However, there is a known problem that it does not have an option to have the changes fully on or off. I just quickly wanted to share my solution as I couldn’t find one online. I redefined the functions add/remove/replace. With this change you can use the function as follows \replace{striked through content}{revised content}. You can also have an optional argument that add bold faced text at the end (e.g., to point to the comment of the reviewer) so \replace[reviewer 1, comment 3]{old content}{new content} appears like this:

Some preceding text. Old Content. New Content [reviewer 1, comment 3]. Text that follows.

Using the command \setreviewsoff then only shows the new content in the manuscript for when you are wanting to have no changes displayed.

Some preceding text. New Content. Text that follows.

Small tip but maybe someone likes it, too. Here the code:

\usepackage[dvipsnames]{xcolor} % for custom colors
\usepackage{easyReview}
\let\oldremove\remove
\renewcommand{\remove}[2][]{\ifistoreview\ifthenelse{\isempty{#1}}
    {\oldremove{\mbox{#2}}}
    {\oldremove{\mbox{#2}} \textbf{[#1]}}%
    \else \fi}
\let\oldadd\add
\renewcommand{\add}[2][]{\ifistoreview\ifthenelse{\isempty{#1}}
    {\oldadd{#2}}
    {\oldadd{#2} \textbf{[#1]}}%
    \else \oldadd{#2}\fi}
\renewcommand{\replace}[3][]{\ifistoreview\ifthenelse{\isempty{#1}}
    {\oldremove{\mbox{#2}} \oldadd{#3}}
    {\oldremove{\mbox{#2}} \oldadd{#3} \textbf{[#1]}}%
    \else \oldadd{#3}\fi}
\renewcommand{\addColor}{\textcolor{OliveGreen}} %OliveGreen

%\setreviewsoff % Comment out if you want the finished version
