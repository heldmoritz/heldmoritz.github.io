I am currently revising a manuscript using easyReview. There is a known problem that there is no option to have the changes on or off. I solved this by redefining the functions add/remove/replace and thought I would just share it with everybody in case someone else is using the package, too. With this change you can use the function as follows \replace{striked through content}{revised content}. You can also have an optional argument that add bold faced text at the end (e.g., to point to the comment of the reviewer) so \replace[reviewer 1, comment 3]{old content}{new content} appears like this:

[!How it appears in text] ![[website_latex_example.png]]

Using the command \setreviewsoff then only shows the new content in the manuscript for when you are wanting to have no changes displayed.

To take over these changes simply include

\usepackage[dvipsnames]{xcolor} % for custom colors
\usepackage{easyReview}
\let\oldremove\remove
\renewcommand{\remove}[2][]{\ifistoreview\ifthenelse{\isempty{#1}}
    {\oldremove{\mbox{#2}}}
    {\oldremove{\mbox{#2}} \textbf{[#1]}}%
    \else \fi}
\let\oldadd\add
\renewcommand{\add}[2][]{\ifistoreview\ifthenelse{\isempty{#1}}
    {\oldadd{#2}}
    {\oldadd{#2} \textbf{[#1]}}%
    \else \oldadd{#2}\fi}
\renewcommand{\replace}[3][]{\ifistoreview\ifthenelse{\isempty{#1}}
    {\oldremove{\mbox{#2}} \oldadd{#3}}
    {\oldremove{\mbox{#2}} \oldadd{#3} \textbf{[#1]}}%
    \else \oldadd{#3}\fi}
\renewcommand{\addColor}{\textcolor{OliveGreen}} %OliveGreen
%\setreviewsoff % Comment out if you want the finished version

Hope this helps!
